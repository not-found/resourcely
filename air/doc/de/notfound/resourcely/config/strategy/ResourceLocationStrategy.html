getOrder()method 
public function getOrder(densities:Vector.<Density>):Vector.<Density>

Classes extending ResourceLocationStrategy should implement this. Resourcely will scan the resource folder in order of the Vector this method returns. It'll use the qualifier to locate the corresponding folders. For example: 'res/ldpi/img.jph' for the the Density with the lowest dpi value.

Parameters

densities:Vector.<Density> — A list of Densities, ordered ascending by dpi value.

Returns
Vector.<Density> — Should return the list in the order resourcely should look through the resource folders.
